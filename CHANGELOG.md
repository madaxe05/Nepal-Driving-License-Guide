## [1.0.0] - 2025-12-28

### Added
- Initial release of Nepal Driving License Guide
- Complete guide for Bike/Scooter license preparation
  - Required documents section
  - Eligibility requirements
  - Application process walkthrough
  - Written exam overview
  - Trial exam details
  - Basic traffic rules
- Complete guide for Car/Jeep license preparation
  - Required documents section
  - Eligibility requirements
  - Detailed application process
  - Comprehensive written exam information
  - Practical trial exam guide
  - Traffic rules and regulations
- Offline functionality - works without internet
- Dark mode support with automatic theme switching
- Clean and modern Material Design UI
- Category-based navigation
- About screen with app information
- Splash screen
- Pro tips and helpful information cards
